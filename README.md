# Trading Watch (ESP32-S3)

Trading Watch is an ESP32-S3-powered market monitoring and trading assistant.

## Project vision
- Fetch real-time market and account data from external APIs
- Monitor markets 24/7 using ESP32-S3 low-power continuous operation
- Send notifications for important market conditions and trading signals
- Execute automated trading actions based on configured strategy logic

## Platform
- Hardware: ESP32-S3
- Firmware framework: Arduino
- Connectivity: Wi-Fi + API integrations

This repository starts as the firmware foundation and will expand with API clients, signal logic, notification modules, and automated trade execution controls.
